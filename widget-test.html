🧪 Widget Test Page

This page is testing the MyAIEmployee embed widget.

✅ The widget should show a floating "S" button
in the bottom-right corner (the first letter of the agent name).
Click it to open the chat panel with the agent's name and role.

Try these attributes on the embed div below:

  • data-agent-id — Your agent ID
  • data-agent-name — Agent display name
  • data-agent-role — Agent role/title
  • data-theme — "dark" or "light"
  • data-agent-options — JSON array of option chips (products/services)
  • data-position — "bottom-right" or "bottom-left"
